## Телеграм бот для преобразования текста в QR код

Присылаете боту текст, в ответ получаете QR код и небольшую информацию о тексте

## Инструкция по запуску
- Создать в директории на удалённом сервере файл docker-compose.yaml
```bash
sudo nano docker-compose.yaml
```
- В него написать следующий текст:
```txt
services:
   bot_qr:
      image: iliamiheev/bot_qr:latest
      container_name: bot_qr
      env_file:
       - .env
```
Далее Ctrl + O, Enter, Ctrl + X
- В этой же директории создать файл с переменными окружения
```bash
sudo nano .env
```
- В него занести единственную переменную — токен от телеграм бота
```txt
API_TOKEN={{ВАШ ТОКЕН}}
```
- Все приготовления завершены, запустить бота
```bash
sudo docker compose up -d
```
- Теперь на машине должен быть запущен контейнер с именем bot_qr. Проверить это можно введя 
```bash
sudo docker ps
```
